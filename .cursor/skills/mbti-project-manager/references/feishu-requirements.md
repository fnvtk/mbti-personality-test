## 1. 项目概述

"神仙团队AI测试系统"是一款基于AI面部分析和心理测试的综合性性格分析工具，通过分析用户的面部特征和心理测试结果，提供MBTI、PDP、DISC等多维度的性格分析报告。第二期开发将优化现有功能，并新增企业版功能，建立完整的支付流程和用户转化链路。

### 1.1 项目背景

第一期开发已完成基础的AI面部分析功能，包括拍照、分析、生成简要报告等功能。第二期需要实现**商业化闭环**，增加支付系统、完善用户转化链路，并开发企业版功能以支持团队测试和简历分析。

### 1.2 目标用户

| 用户类型 | 画像 |
|:---|:---|
| **个人用户** | 求职者、职场人士、对自我认知有需求的个人 |
| **企业用户** | 创业团队、中小企业、HR部门、团队管理者 |

---

## 2. 系统架构

```
用户端（小程序/H5）
  ├── 个人版：拍照→AI分析→简要报告→付费→完整报告→微信添加
  └── 企业版：咨询→提交需求→确认方案→支付→系统设置→团队分析
后端（Next.js API Routes）
  ├── 认证/支付/订单
  ├── AI分析（Groq/Coze）
  └── 企业管理
数据库（PostgreSQL + MongoDB）
```

---

## 3. 核心功能需求

### 3.1 关键操作点优化

| 操作点 | 要求 |
|:---|:---|
| **拍照流程** | 三张不同角度照片（正面、左侧45度、右侧45度） |
| **登录卡点** | 用户必须登录才能查看报告 |
| **微信引导** | 系统通过存客宝自动添加用户微信 |
| **结果展示** | 简要报告页面展示基础结果，**锁定**高级内容 |
| **付费引导** | 明确的付费入口，显示锁定的高级内容预览 |

### 3.2 支付系统实现

#### 3.2.1 普通版（个人版）

| 项目 | 价格 |
|:---|:---:|
| 基础报告 | ¥198/次 |
| 深度报告 | ¥1,980/次 |

- **支付方式**：微信支付、支付宝
- **支付流程**：选择服务 → 确认订单 → 支付 → 微信添加并服务

#### 3.2.2 企业版

| 套餐 | 价格 | 团队规模 |
|:---|:---:|:---:|
| 团队启动版 | ¥19,800 | 10人 |
| 团队成长版 | ¥39,800 | 10-30人 |
| 团队蜕变版 | ¥98,000 | 最多100人 |

- **支付方式**：微信支付、支付宝、对公转账
- **企业版支付流程**：咨询 → 提交需求 → 确认方案 → 支付 → 系统设置

### 3.3 企业版功能开发

#### 3.3.1 简历上传功能
- 首页增加"上传简历"按钮（企业版专属）
- 支持 PDF、Word 格式简历上传
- AI分析简历与面部特征的匹配度
- 生成求职者综合评估报告

#### 3.3.2 团队分析功能
- 团队成员性格组合分析
- 团队潜在冲突点分析
- 团队优势与短板评估
- 团队角色建议与优化方案

#### 3.3.3 企业管理后台
- 员工/应聘者管理
- 测试结果统一查看
- 团队分析报告生成
- 权限管理功能

### 3.4 版本控制系统

#### 3.4.1 超级管理员功能
- 控制系统显示为个人版或企业版
- 功能模块开关控制
- 用户权限管理
- 系统参数配置

## 5. 用户转化链路（核心商业逻辑）

```
免费体验（拍照+基础分析）
    ↓
登录卡点（必须登录查看报告）
    ↓
简要报告（基础结果 + 锁定高级内容预览）
    ↓
付费引导（¥198基础 / ¥1,980深度）
    ↓
完整报告 + 存客宝自动添加微信 + 持续服务
    ↓
企业版升级引导（团队测试/简历分析）
```
