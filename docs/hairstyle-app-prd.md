# 发型魔镜 - 产品需求文档

## 一、产品概述

### 产品名称
发型魔镜

### 一句话定义
用户上传一张真实近照，系统基于脸型自动生成适合他的发型效果图，并支持付费生成与分销裂变。

---

## 二、产品目标

- 7天内完成 MVP，可真实使用
- 跑通：上传 → AI生成 → 付费 → 出图 → 分销
- 单点变现，不做复杂功能

---

## 三、核心用户

1. **普通用户**：想换发型但不敢剪
2. **设计师 / 理发店**：设计辅助工具

---

## 四、用户核心路径

1. 打开小程序 / App
2. 上传近照
3. 系统识别脸型 + 推荐发型
4. 选择生成 1–4 张
5. 支付（5元/张）
6. 查看结果
7. 引导注册 / 分享分销

---

## 五、功能模块

### 5.1 照片上传模块
- 支持拍照 / 相册
- 校验正脸、无遮挡
- 上传后本地缓存

### 5.2 AI 换发型生成模块（核心）
- 使用模型：Nano Banana Pro（图像生成）+ ChatGPT（脸型判断）
- 流程：图片 → 脸型分析 → 输出脸型标签 → 匹配发型 Prompt → 调用生成接口

### 5.3 发型库模块
- 发型分类：男/女、长/中/短
- 点击即替换
- 每个发型 = 一个 Prompt 模板

### 5.4 付费模块
- 价格：5元/张
- 生成前或查看前付费
- 支付方式：微信/支付宝

### 5.5 登录 & 用户系统
- 登录方式：手机验证码
- 登录触发：首次生成完成后
- 用户数据：历史生成记录、订单记录

### 5.6 分销系统
- 自动生成分销 ID
- 分享链接 / 卡片
- 一级分润 50%
- 自动结算逻辑
